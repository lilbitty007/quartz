#### **方法 1：使用 HTML + CSS（通用方法）**
```
<span style="color:red">这段文字是红色的</span>
<span style="color:blue">这段文字是蓝色的</span>
<span style="color:#FF5733">这段文字是橙色的（十六进制）</span>
```

- 渲染

<span style="color:red">这段文字是红色的</span>
<span style="color:blue">这段文字是蓝色的</span>
<span style="color:#FF5733">这段文字是橙色的（十六进制）</span>



___



#### 方法 2：使用 Markdown 扩展
例如：

- **Obsidian** 支持 ==高亮==（黄色高亮），但不支持直接改颜色。

```
==颜色==
```

- 渲染

==颜色==

___




#### **方法 3：使用代码块 + 语言高亮（仅限代码）**

```html
<font color="red">这段文字是红色的</font>

```


```js
function hello() {
  console.log("Hello, world!");
}
```


> ```text
> <span style="color:orange;">这是橙色</span> 普通文本
> ```


```python {highlightLines="2"}
# 普通注释
print("Hello")  # 高亮行
```




```
<span style="color: red;">这是红色文字</span>
<span style="color: green;">这是绿色文字</span>
```









---


#### **方法 4：使用 LaTeX（数学公式变色）**

在支持 LaTeX 的 Markdown 环境（如 Jupyter Notebook、Typora）中，可以用 \color 命令改变公式颜色：

```
$\color{red}{红色文字}$  $\color{blue}{蓝色文字}$
```
- **渲染效果：**  

$\color{red}{红色文字}$  $\color{blue}{蓝色文字}$


蓝色文字蓝色文字

✅ **适用场景**：数学公式、科学笔记。  
❌ **不适用**：普通文本。



#### **总结**

|方法|适用场景|兼容性|备注|
|---|---|---|---|
|**HTML + CSS**|通用文本变色|高（支持 HTML 的平台）|最推荐|
|**Markdown 扩展**|特定平台（如 Obsidian）|中|通常只能高亮，不能自定义颜色|
|**代码块高亮**|代码关键字|高|仅限代码|
|**LaTeX**|数学公式|中|仅限公式|
